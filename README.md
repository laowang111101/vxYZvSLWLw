# 企业员工管理系统 python969

## 项目概述

企业员工管理系统是一个全面的管理工具，旨在通过高效的信息管理提高企业内部工作效率。系统提供部门管理、用户管理、任务分配等多种功能，确保企业运营数据的有效整合与管理。

## 技术栈

- **前端**：HTML、CSS、JavaScript、jQuery
- **后端**：Python、Django
- **数据库**：MySQL
- **开发工具**：VSCode、Navicat

## 系统角色

- **普通用户**：能够管理个人信息、查看和完成任务等。
- **管理员**：负责部门管理、用户管理、设备管理以及订单数据的可视化。

## 运行环境

- Python 3.0 或更高版本
- MySQL 8.0 或更高版本

## 功能模块

- **部门管理**：管理企业内各个部门的信息。
- **用户管理**：对员工信息进行添加、删除、修改和查询。
- **订单管理**：处理订单相关信息。
- **任务管理**：分配、跟踪和报告任务进度。
- **手机管理**：管理员工手机设备信息。
- **设备管理**：对企业设备进行管理。
- **管理员管理**：对管理员权限进行设置和管理。
- **订单数据可视化**：提供订单数据的可视化展示。
- **修改个人信息**：允许用户更新个人信息。
